# Fake Product Review Detection Using NLP

## Project Description

This project detects whether a product review is genuine or fake using Natural Language Processing (NLP) and Machine Learning.

## Technologies Used

- HTML
- CSS
- JavaScript
- Python
- Pandas
- NumPy
- Scikit-Learn
- Logistic Regression

## Features

- Text preprocessing
- TF-IDF vectorization
- Logistic Regression classifier
- Real-time review prediction

## How to Run

pip install -r requirements.txt

python train_model.py

python app.py
